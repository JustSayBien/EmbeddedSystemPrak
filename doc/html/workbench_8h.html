uint8_t  id_to 
)

Get the angle between two workbenches.

This function returns the angle between the workbenches. It either gets the previously calculated angle value out of the 'cache' or calculates the angle and stores it in the ’cache’ for the next time.

Parameters:
id_fromthe id of the current base
id_tothe id of the base to drive to
Returns:
the angle between the the specified workbenches. The value is in degrees and signed (+ counter clockwise, - clockwise)
